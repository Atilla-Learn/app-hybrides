Une application que vous connaissez peut être
Quelques caractéristiques
  • Python

  • Open Source
  • Licence MIT
Hello World
						
from kivy.app import App
from kivy.uix.button import Button

class TestApp(App):
  def build(self):
      return Button(text='Hello World')

TestApp().run()
Comment ça marche?
  • Widgets
  • Système de callback
  • Description graphique dans un .kv
Exemple d'un .kv
						
#:kivy 1.9.0

<DelEventPopup>:
    size_hint: (0.3, 0.3)
    title: 'Voulez vous vraiment supprimer cet évènement?'
    title_size: 15

    BoxLayout:
        spacing: 20
        padding: 20
        Button:
            text: "Oui"
            on_release: root.delete_event()
        Button:
            text: "Non"
            on_release: root.dismiss()
Les avantages
  • Langage haut niveau
  • Dévelopement simplifié (pas d'émulateur)
  • Accès à l'éco-système Python
  • Déploiement très simple
    • 							
	buildozer android debug deploy run
Les inconvénients
  • Quelques problèmes de documentation
  • La communauté n'est pas très grande
  • Encore quelques bugs pour le passage sous Android?
  • Assez récent pour iOS
  • Une GUI pour créer l'interface est en cours
